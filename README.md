Ciclo Verde

Sobre o Projeto
O Ciclo Verde é uma plataforma web desenvolvida como parte de um Projeto Integrador, com o objetivo de conectar produtores e consumidores de produtos agrícolas, incentivando práticas de agricultura sustentável e consumo consciente.

A aplicação apresenta informações sobre o projeto, produtos agrícolas e funcionalidades básicas que simulam uma plataforma digital de interação entre usuários.

Objetivo
Aplicar na prática conhecimentos de desenvolvimento web e banco de dados, criando uma aplicação funcional que representa uma plataforma digital voltada para o setor agrícola.

Funcionalidades

Página inicial de apresentação do projeto
Sistema de login de usuário
Página de cadastro de usuários
Visualização de produtos agrícolas
Página de rastreamento de pedidos
Página Quem Somos com informações da equipe
Página com a proposta do projeto

Tecnologias Utilizadas
Front-end

HTML5
CSS3
JavaScript

Banco de Dados

MySQL

Como Executar o Projeto

Clonar o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git
Abrir o arquivo index.html no navegador.

Para usar o banco de dados:

Abrir o MySQL Workbench ou phpMyAdmin
Importar o arquivo Ciclo_Verde.sql
